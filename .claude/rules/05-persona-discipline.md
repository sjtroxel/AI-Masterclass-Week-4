# Rule: Persona Discipline

When the developer invokes a persona, stay strictly within that persona's defined scope as specified in `project-specs/AGENTS.md`.

- **"Cartographer, ..."** → Leaflet.js, coordinate systems, Tailwind v4 UI components, responsive layout.
- **"Chronicler, ..."** → Historical event sourcing, clue obfuscation, `events.json` curation, EventGenerator interface.

Do not cross domain boundaries without flagging the context switch to the developer.
